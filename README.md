## Scripts

### gather_tree.py

A Python script that gathers the tree structure of a folder and saves it to a JSON file.

**Usage:**
```bash
python scripts/gather_tree.py <folder_path> <output_json_path>
```

**Features:**
- Recursively scans directory structures
- Follows symlinks (with cycle detection)
- Includes hidden files
- Outputs clean JSON structure with file/directory names and hierarchy
- Provides logging and execution reports

For more details, see `docs-vibe/01-folder-tree-to-json.md`.

### rebuild_tree.py

A Python script that rebuilds the tree structure of a folder from a JSON file generated by `gather_tree.py`.

**Usage:**
```bash
python scripts/rebuild_tree.py <json_path> <target_path>
```

**Features:**
- Reads JSON files generated by `gather_tree.py`
- Recreates directory structure recursively
- Creates empty files matching the original structure
- Fail-fast error handling for conflicts
- Progress bar for large trees
- Provides logging and execution reports

For more details, see `docs-vibe/02-json-to-folder-tree.md`.
